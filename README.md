# 🏥 Hospital Analytics Dashboard (SQL + Power BI)

## Project Overview

I built this project to practice SQL and Power BI using a hospital management dataset. My goal was to solve real business problems with SQL and then create an interactive dashboard in Power BI.

This project helped me understand how SQL and Power BI work together in a real-world data analytics workflow.

---

## Tools Used

- MySQL
- Power BI
- Power Query
- DAX

---

## Dataset

The project is based on a hospital management dataset with the following tables:

- Patients
- Doctors
- Appointments
- Treatments
- Billing

---

## SQL Work

In this project, I practiced different SQL concepts such as:

- Joins
- Aggregate Functions
- GROUP BY & HAVING
- Subqueries
- Window Functions
- CASE WHEN
- CTEs
- Business Reporting Queries

Some of the business questions I solved are:

- Top 5 doctors by appointments
- Department-wise revenue
- Revenue contribution percentage
- Patients who never scheduled an appointment
- Patients spending more than the average
- Top revenue-generating doctors
- Patients visiting multiple departments
- Monthly revenue trend

---

## Power BI Dashboard

The dashboard includes:

- Total Revenue
- Total Patients
- Total Appointments
- Total Doctors
- Average Bill Amount
- Monthly Revenue Trend
- Revenue by Department
- Patients by Gender
- Patients by Age Group
- Patients by Blood Group
- Top 5 Doctors by Revenue
- Interactive Slicers (Date, Department and Doctor)

---

## Project Workflow

1. Imported the dataset into MySQL.
2. Wrote SQL queries to answer business questions.
3. Imported the cleaned data into Power BI.
4. Created relationships between tables.
5. Built DAX measures.
6. Designed an interactive dashboard.
